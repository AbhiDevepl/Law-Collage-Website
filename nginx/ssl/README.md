# NGINX SSL Certificate Directory

This directory contains SSL certificates used by NGINX for HTTPS/TLS termination.

## Certificate Types

### Development (Self-Signed)
- **Files:** `cert.pem`, `key.pem`
- **Generated by:** `scripts/generate-self-signed.sh`
- **Validity:** 365 days (customizable)
- **Use case:** Local testing and development

### Production (Let's Encrypt)
- **Files:** Symlinked from `/etc/letsencrypt/live/<domain>/`
- **Generated by:** `certbot` command
- **Validity:** 90 days (auto-renewed)
- **Use case:** Production deployments

## File Permissions

```
-rw-r--r-- 1 cert.pem   (World readable - public certificate)
-r------ 1 key.pem     (Private key - only readable by owner)
```

⚠️ **IMPORTANT:** Never commit private keys to version control!
The `.gitignore` already excludes this directory.

## Usage

### Generate Self-Signed Certificates

```bash
chmod +x ../scripts/generate-self-signed.sh
../scripts/generate-self-signed.sh localhost 365
```

### Use Let's Encrypt Certificates

```bash
sudo certbot certonly --standalone -d yourdomain.com
sudo cp /etc/letsencrypt/live/yourdomain.com/fullchain.pem ./cert.pem
sudo cp /etc/letsencrypt/live/yourdomain.com/privkey.pem ./key.pem
sudo chown $(id -u):$(id -g) *.pem
```

## Docker Volume Mount

In `docker-compose.yml`:

```yaml
volumes:
  - ./nginx/ssl:/etc/nginx/ssl:ro
```

The `:ro` flag makes it read-only inside the container.

## Renewal and Rotation

For Let's Encrypt certificates:

```bash
# Test renewal
sudo certbot renew --dry-run

# Perform renewal
sudo certbot renew

# Reload NGINX after renewal
docker exec ssnlc-nginx nginx -s reload
```

## Troubleshooting

### Certificate not found
```bash
ls -la
# Verify cert.pem and key.pem exist
```

### NGINX won't start
```bash
# Check NGINX logs
docker logs ssnlc-nginx

# Verify certificate syntax
openssl x509 -in cert.pem -text -noout
```

### Permission denied
```bash
# Fix permissions
chmod 644 cert.pem
chmod 600 key.pem
```

See [SSL-SETUP-GUIDE.md](../SSL-SETUP-GUIDE.md) for complete documentation.
